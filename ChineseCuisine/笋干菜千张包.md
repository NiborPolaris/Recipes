# 笋干菜千张包

## 材料

| 材料   | 数量 |
| ------ | ---- |
| 笋干菜 | 100g |
| 虾仁   | 10个 |
| 五花肉 | 500g |
| 千张   | 12张 |
| 鸡蛋   | 1枚  |
| 小葱   | 少许 |
| 姜     | 少许 |

## 做法

1. 五花肉去皮切丁剁成馅，入料酒、葱花、姜末、盐调味。
2. 千张切成大方块，入肉馅包成枕头包，以鸡蛋液封口；每个千张包所用肉馅约50克。
3. 将包好的千张包上屉蒸15分钟。
4. 温水泡开笋干菜，依个人口感切段或切末，入锅煸香；加水烹煮至笋干菜变软。
5. 入千张包和虾仁，再烹煮4分钟；加少许盐、糖调味，开大火收汁。俟成为半汤菜即可出锅。

## 注记
- 这菜适合春天做。为保持清新口感，请不要加酱油。
- 笋干菜系将梅菜与笋片一同烘干的制成品，可直接用梅干菜代替。
