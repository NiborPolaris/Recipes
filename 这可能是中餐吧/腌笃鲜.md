# 腌笃鲜

## 材料

| 材料               | 数量        |
| ----------------- | ----------- |
| 五花肉             | 300g        |
| 咸肉			   | 200g        |
| 火腿               | 50g         |
| 冬笋或春笋          | 1棵         |
| 千张               | 6张         |
| 小葱               | 少许        |
| 姜                 | 少许        |

## 做法

1. 五花肉、咸肉切小块，火腿切片；焯水，撇去浮沫。
2. 千张切条，沿长边对折卷起，挽成百叶结，焯水去除豆腥味；冬笋或春笋去壳，焯水去除涩味，切滚刀块。
3. 起汤锅加足以没过食材的水，加入葱结姜片；入五花肉、咸肉、火腿，中火炖煮1小时。
4. 俟汤汁变成奶白色，再加入百叶结和笋块，炖煮30分钟，拣出葱结姜片，撒上葱花。

## 注记

- 腌笃鲜是「汤菜」而非「汤」。盛盘时，有足量的食材和适量的汤水，观感最佳。
- 几乎无需额外调味。
